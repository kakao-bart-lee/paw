# Backlog ↔ Back Office 영향 매핑

이 문서는 제품 백로그 항목을 백오피스 요구사항과 연결해 추적합니다.

## 상태 정의

- `planned`: 준비 필요
- `in_progress`: 구현 중
- `done`: 구현 + 테스트 + 문서 반영 완료
