# Paw 앱 디자인 요청

안녕하세요.  
Paw 모바일 앱의 UI/UX 디자인을 의뢰드립니다.

## Paw가 어떤 서비스인지

Paw는 **사람과 AI Agent가 같은 대화방 안에서 함께 대화하고 협업할 수 있는 메신저**입니다.

일반 메신저처럼 사람끼리 메시지를 주고받을 수 있고, 동시에 AI Agent를 대화에 참여시켜서 질문하고, 답변을 받고, 작업을 도와받는 구조를 목표로 하고 있습니다.

즉, Paw는 단순 채팅 앱이 아니라:
- 사람 ↔ 사람 대화
- 사람 ↔ AI Agent 협업
- 그룹 대화
- 보안 대화
- 향후 Agent 탐색/설치
를 하나의 앱 안에서 자연스럽게 연결하는 서비스입니다.

## 현재 파악된 핵심 기능

현재 앱 구조와 코드 기준으로, 주요 기능은 아래와 같습니다.

### 1. 로그인 / 계정
- 전화번호 입력
- OTP 인증
- 디바이스 등록
- username 설정
- 로그인 상태 복구

### 2. 채팅
- 대화 목록
- 채팅방 상세
- 메시지 전송
- 그룹 대화
- 검색
- 읽지 않음, 입력 중, 상태 표시

### 3. AI Agent 기능
- Agent가 대화에 참여 가능
- Agent 응답 표시
- 실시간 스트리밍 응답
- Agent 권한 동의
- 향후 Agent 목록/탐색/설치 확장 예정

### 4. 보안 기능
- 보안 대화(E2EE) 상태 표시
- 키 검증 관련 흐름
- 디바이스/세션 개념
- 향후 차단, 신고, 세션 관리 확장 예정

### 5. 설정 / 프로필
- 내 프로필
- username 수정
- 알림 / 개인정보 / 보안 설정
- 도움말 / 지원

## 디자인에서 특히 중요하게 보고 싶은 점

이번 디자인에서는 단순히 화면을 예쁘게 만드는 것보다, 아래를 중요하게 반영해주시면 좋겠습니다.

- **메신저답게 빠르고 직관적인 구조**
- **AI 기능이 억지스럽지 않고 자연스럽게 녹아드는 경험**
- **보안 기능이 어렵지 않게 이해되는 흐름**
- **모바일에서 실제로 쓰기 편한 사용성**
- **한국어 중심의 좋은 가독성**

## 원하는 앱 분위기

현재 제품 톤은 대체로:
- 다크 모드 중심
- 따뜻한 느낌의 딥한 컬러 톤
- 너무 가볍지 않은, 차분하고 신뢰감 있는 분위기
- 메신저이지만 약간 도구적이고 세련된 느낌

방향으로 생각하고 있습니다.

즉,
**“보안 메신저 + AI 협업 도구”** 같은 인상을 주는 모바일 앱이면 좋겠습니다.

## 요청드리고 싶은 디자인 범위

아래 화면/흐름을 설계 부탁드립니다.

- 로그인 흐름
- 대화 목록
- 채팅방 상세
- 새 대화 시작 / 그룹 생성
- Agent 관련 화면
- 프로필 / 설정
- 보안 관련 화면 또는 상태 표현

필요하다면 empty / loading / error 상태까지 함께 제안 부탁드립니다.

## 최종 요청

정리하면,  
**Paw를 “사람과 AI가 함께 대화하고 협업하는 보안 메신저”로 보고, 모바일 앱 UI/UX를 설계해 주세요.**

기존 메신저의 익숙함은 유지하되,
AI Agent, 보안, 실시간 응답이라는 특징이 자연스럽게 드러나는 방향을 기대합니다.
