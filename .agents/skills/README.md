# PAW Project Skill Bundle

This project-scoped skill bundle is tailored for PAW's current stack:

- Rust workspace: `paw-core`, `paw-server`, `paw-proto`, `paw-crypto`
- Android: Kotlin + Jetpack Compose in `paw-android`
- iOS: SwiftUI in `paw-ios`
- Flutter Web/Desktop in `paw-client`
- Verification lanes: API, performance, accessibility, integration readiness

## Installed Skills

- `Mobile App Builder`: Primary agent for Android, iOS, and Flutter client work
- `Frontend Developer`: Best fit for Flutter Web/Desktop UI, interaction, and performance work
- `Backend Architect`: Best fit for Rust/Axum server APIs and backend contracts
- `Software Architect`: Best fit for cross-platform boundaries, UniFFI contracts, and migration decisions
- `Security Engineer`: Best fit for auth, crypto, secure storage, and messaging trust boundaries
- `Code Reviewer`: Best fit for mixed-language PR and patch review
- `Technical Writer`: Best fit for README, migration docs, runbooks, and API docs
- `SRE`: Best fit for reliability, observability, deployment, and operational guardrails
- `API Tester`: Best fit for server/API and contract verification
- `Performance Benchmarker`: Best fit for runtime, API, and app performance bottlenecks
- `Accessibility Auditor`: Best fit for Flutter, Compose, and SwiftUI accessibility checks
- `Reality Checker`: Best fit for final readiness checks before calling work complete

## Recommended Pairing

Use these with the existing OMX workflow skills already available in your environment:

- `plan`: for multi-surface execution plans
- `team`: for parallel Android/iOS/Rust/Flutter work
- `code-review`: for review workflow orchestration
- `security-review`: for explicit security passes
- `visual-verdict`: for screenshot-based UI checks
