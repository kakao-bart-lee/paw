# Paw Skills

Project-specific skills for the Paw Rust server and protocol workspace.

| Skill | Description | When to Use |
|-------|-------------|-------------|
| **rust-backend** | Axum handlers, SQLx queries, auth middleware, WebSocket hub | Adding or modifying REST endpoints, database queries, or real-time features in `paw-server/` |
| **migration-author** | PostgreSQL migration authoring with Paw timestamp conventions | Creating or reviewing schema changes in `paw-server/migrations/` |
| **protocol-designer** | paw-proto WebSocket message types with backward compatibility | Adding or modifying WebSocket protocol messages in `paw-proto/` |
